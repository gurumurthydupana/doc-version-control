# Intelligent Document Version Control System

This project is a MERN Stack application that provides AI-powered document version control.

## ✅ Features

- User Authentication using JWT (Editor / Viewer Roles)
- Document Creation and Editing
- Version Saving with Metadata (timestamp, user)
- AI-generated Change Summaries (Mock AI integrated)
- Version History List
- Rollback to Any Previous Version

---

## ✅ Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB + Mongoose
- API Testing: Thunder Client / Postman
- AI Summary: OpenAI (Mocked Implementation)
